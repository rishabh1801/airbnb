# Airbnb Scraper

A Scrapy-based scraper for Airbnb listings that sends data to a Django backend.

## Features

- Scrapes detailed information about Airbnb listings
- Handles pagination to get multiple pages of results
- Identifies and uses Airbnb's API endpoints
- Sends scraped data to a Django backend

## Requirements

- Python 3.7+
- Scrapy
- Requests
- (Optional) Selenium and ChromeDriver for JavaScript rendering
